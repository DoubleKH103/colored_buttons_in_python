# Colored Buttons in PyQt5 

A simple **PyQt5 desktop application** that demonstrates how to create and style buttons using **Qt Stylesheets (QSS)**.  
The app displays three colored buttons arranged horizontally, each with hover effects and custom styling.

## Preview
The application window contains three buttons:
- Red button (`#1`)
- Green button (`#2`)
- Yellow button (`#3`)

Each button changes color when hovered.

## Features
- PyQt5 GUI application
- Horizontal layout using `QHBoxLayout`
- Custom button styling with Qt Stylesheets
- Hover effects
- Clean and simple structure
